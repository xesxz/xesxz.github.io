# vue

## VUE 装饰器

vue 装饰器有两种
vue-class-component(官方版)

```js
import Vue from "vue";
import Component from "vue-class-component";
```

vue-property-decorator(社区版)

```js
import { Component, Vue } from "vue-property-decorator";
```
